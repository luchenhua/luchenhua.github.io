Brand New Start

April 1, 2017
hexo github-page

个人

利用20%时间看着官方的文档，在 GitHub 上利用 GitHub Pages 和 Hexo 搭建了个人博客。

博客也就用来写写想法、心得、收货、笔记之类的东西，不是心血来潮，而是觉得做为一个软件公司的程序员，尤其在创业公司内折腾过很多事情，挖过也踩过很多坑，如果不记下来的话自己做为一个程序员以及 leader 是失职的。

Hexo 在搭建过程中没遇到太大的麻烦，倒是在换 Material Theme 的时候有点脑子抽风了。应该写在 site config 的 _config.yml 里的 language: zh-CN ，一直尝试着往 theme config 的 _config.yml 里写，然后在 hexo generate 的时候又不报错，但是 hexo server 启动以后一直是显示的阿拉伯文 Orz。反复折腾了几遍又研究了一下 Hexo 以及 Material Theme 的官方文档，才意识到 Material Theme 的官方文档中标注的 BasicConfig 其实就是指 Hexo 的 site config ，至此问题解决。

团队

话说最近在反复思考自己到底要怎样的团队，自己该做些什么才能带出这样的团队，自己首先要做到些什么，自己想要的风格是什么，团队的风格是什么，做为 leader 要怎样服务于团队和组员，怎样才能给组员带来职业上的成长。

挺苦恼的，挺纠结的，也挺开心的。这事情还是要自己探索扪心自问吧，反复不断的格物格自己，以求去伪存真。

结论

Anyway ，喜欢现在的公司，喜欢现在的组员和同事们，喜欢现在的责任，喜欢现在的生活。

想要成长，变得更好更可靠。

The End
